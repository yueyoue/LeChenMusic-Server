# LeChenMusic — 完整开发文档

> 最后更新: 2026-07-22

## 1. 项目概述

### 1.1 产品定位

LeChenMusic 是一个 **音乐 + 有声书 + 影视** 一体化的个人流媒体平台，包含：

- **服务端**：基于 Navidrome fork（Go + SQLite），提供音乐/有声书的扫描、存储、API 服务
- **APP 端**：Kotlin + Jetpack Compose 的 Android 客户端，对接服务端 + LunaTV 影视服务

### 1.2 仓库地址

| 仓库 | 地址 | 说明 |
|------|------|------|
| 服务端 | https://github.com/yueyoue/LeChenMusic-Server | Go, 基于 Navidrome fork |
| APP 端 | https://github.com/yueyoue/LeChenMusic-Player | Kotlin, Jetpack Compose |
| LunaTV | https://github.com/MoonTechLab/LunaTV | 影视服务端（独立部署） |
| Selene-Source | https://github.com/MoonTechLab/Selene-Source | LunaTV 配套 Flutter APP（影视功能参考） |

### 1.3 技术栈

| 层 | 服务端 | APP 端 |
|---|---|---|
| 语言 | Go | Kotlin |
| UI | React + Material-UI (Web 管理后台) | Jetpack Compose + Material 3 |
| 数据库 | SQLite | — |
| API | Subsonic API + 自定义 REST | Retrofit + OkHttp |
| 播放器 | — | Media3 ExoPlayer |
| 图片 | — | Coil |
| 部署 | Docker | APK (minSdk 26, targetSdk 34) |

### 1.4 功能模块总览

| 模块 | 服务端 API | APP 端页面 | 数据来源 |
|------|-----------|-----------|---------|
| 🎵 音乐 | Subsonic API + REST | 首页/专辑/歌手/歌曲/播放器 | 服务端扫描本地文件 |
| 📖 有声书 | 自定义 REST API | 首页(有声书模式)/列表/详情/播放器 | 服务端扫描本地文件 |
| 🎬 影视 | LunaTV API (外部) | 首页/搜索/详情/播放器/分类/直播 | LunaTV + 豆瓣 |
| 📻 电台 | Subsonic API | 电台页 | 服务端 |
| 📋 歌单 | Subsonic API | 歌单列表/详情 | 服务端 |
| ⭐ 收藏 | Subsonic API + 自定义 | 收藏页 | 服务端 |
| 🔍 搜索 | Subsonic API + LunaTV | 搜索页 | 服务端 + LunaTV |

## 2. 服务端 API

### 2.1 认证方式

所有 API 请求需要携带认证参数：

| 参数 | 说明 |
|------|------|
| `u` | 用户名 |
| `p` | 密码（enc:hex 编码格式，如 `enc:656e63`） |
| `t` | Token（替代密码认证） |
| `v` | API 版本，固定 `1.16.1` |
| `c` | 客户端名称，如 `lechenmusic` |
| `f` | 返回格式，固定 `json` |

**示例：**
```
GET /api/audiobook?u=admin&p=enc:xxxx&v=1.16.1&c=lechenmusic&f=json
```

### 2.2 有声书 API

> 自定义 REST API，非 Subsonic 标准。

| 端点 | 方法 | 说明 | 请求参数 | 返回格式 |
|------|------|------|---------|---------|
| `/api/audiobook` | GET | 有声书列表 | `?genre=评书&narrator=艾宝良` | `{audiobooks: [...]}` |
| `/api/audiobook/genres` | GET | 分类列表及各分类数量 | — | `{genres: [{name, count}]}` |
| `/api/audiobook/narrators` | GET | 演播者列表及作品数 | — | `{narrators: [{name, count}]}` |
| `/api/audiobook/starred` | GET | 收藏的有声书 | — | `{audiobooks: [...]}` |
| `/api/audiobook/:id` | GET | 有声书详情（含章节列表 + 当前用户进度） | — | `{book, chapters, progress}` |
| `/api/audiobook/:id/chapters` | GET | 章节列表 | — | `{chapters: [...]}` |
| `/api/audiobook/:id/chapters/:cid/stream` | GET | 流式播放章节 | — | 音频二进制流 |
| `/api/audiobook/:id/progress` | GET | 获取播放进度 | — | `{progress}` |
| `/api/audiobook/:id/progress` | PUT | 更新播放进度 | body: `{chapterId, position, playbackSpeed, skipIntro, skipOutro}` | `{ok: true}` |
| `/api/audiobook/:id/bookmark` | GET | 获取书签列表 | — | `{bookmarks: [...]}` |
| `/api/audiobook/:id/bookmark` | POST | 添加书签 | body: `{chapterId, position, title}` | `{ok: true}` |
| `/api/audiobook/:id/bookmark/:bid` | DELETE | 删除书签 | — | `{ok: true}` |
| `/api/audiobook/:id/star` | POST | 收藏 | — | `{ok: true}` |
| `/api/audiobook/:id/star` | DELETE | 取消收藏 | — | `{ok: true}` |
| `/api/audiobook/:id/cover` | GET | 封面图 | — | 图片二进制 |

```json
  "narrator": "艾宝良",

  "year": 2020,
  "coverPath": "cover.jpg",
  "totalDuration": 101700,
  "chapterCount": 120,
  "series": "",

  "path": "有声读物/三体",
  "size": 536870912,
  "createdAt": "2026-07-01T00:00:00Z",
  "updatedAt": "2026-07-01T00:00:00Z"
}
```

**章节对象结构：**
```json
{

  "audiobookId": "string",
  "title": "01",
```


**Genre 枚举值：** `有声读物` / `评书` / `相声` / `戏曲` / `儿童` / `教育`

### 2.3 音乐 API (Subsonic 标准)

沿用 Navidrome / Subsonic API，核心端点：

| 端点 | 说明 |
|------|------|
| `/rest/getAlbumList2` | 专辑列表（支持 `newest/random/recent/frequent/starred` 类型） |
| `/rest/getAlbum` | 专辑详情 |
| `/rest/getArtist` | 艺术家详情 |
| `/rest/getArtists` | 艺术家列表 |

| `/rest/stream` | 流式播放 |
| `/rest/getStarred` | 收藏列表 |
| `/rest/star` / `unstar` | 收藏/取消收藏 |
| `/rest/getPlaylists` | 歌单列表 |
| `/rest/getPlaylist` | 歌单详情 |
| `/rest/createPlaylist` | 创建歌单 |
| `/rest/updatePlaylist` | 更新歌单 |
| `/rest/deletePlaylist` | 删除歌单 |
| `/rest/getSong` | 歌曲详情 |
| `/rest/getRandomSongs` | 随机歌曲 |
| `/rest/getNowPlaying` | 正在播放 |
| `/rest/getRecentSongs` | 最近播放 |

| `/rest/getInternetRadioStations` | 电台列表 |
| `/rest/getLyrics` | 歌词 |

**Subsonic 响应格式（统一包装）：**
```json
{
  "subsonic-response": {
    "status": "ok",
    "version": "1.16.1",
    // ... 具体数据
  }
}
```

### 2.4 影视 API (LunaTV)

> LunaTV 是独立部署的服务，APP 端直接对接，不经过 LeChenMusic Server。

**LunaTV 服务地址：** `http://j.tthsdd.top:3000`

⚠️ **LunaTV 返回裸 JSON，没有 code/msg/data 包装！**

| 端点 | 方法 | 说明 | 请求参数 | 返回格式 |
|------|------|------|---------|---------|
| `/api/login` | POST | 登录 | body: `{"username":"admin","password":"***"}` | `{"ok": true}` |
| `/api/search` | GET | 搜索影片 | `?q=关键词` ⚠️ 参数是 `q` 不是 `keyword` | `{"results": [VideoInfo...]}` |
| `/api/detail` | GET | 影片详情 | `?source=xxx&id=xxx` | `VideoDetail` 裸对象 |
| `/api/playrecords` | GET | 播放记录 | — | `{"source+id": {record...}}` ⚠️ 是 dict 不是 list |
| `/api/playrecords` | POST | 保存播放记录 | body: `{"key":"source+id", "record":{...}}` | — |
| `/api/favorites` | GET | 收藏列表 | — | `[VideoInfo...]` 裸数组 |
| `/api/search/resources` | GET | 片源列表 | — | `[SearchResource...]` 裸数组 |
| `/api/live/sources` | GET | 直播源列表 | — | `[LiveSource...]` 裸数组 |
| `/api/live/channels` | GET | 直播频道列表 | — | `[LiveChannelGroup...]` 裸数组 |

⚠️ **不存在的接口**（返回 404）：`/api/home`、`/api/category`

**VideoInfo 对象结构：**
```json
{
  "id": "65490",
  "title": "痴迷2025",
  "poster": "https://...",
  "episodes": ["https://xxx.m3u8", "https://xxx.m3u8", ...],
  "episodes_titles": ["1", "2", ...],
  "source": "iqiyizyapi.com",
  "source_name": "🎬-爱奇艺-",
  "year": "2025",
  "type_name": "movie",
  "douban_id": 0
}
```

**VideoDetail 对象结构：** 包含完整的影片信息 + 所有集数 URL + 所有可用源

### 2.5 豆瓣 API (影视推荐)

| 端点 | 说明 | 注意事项 |
|------|------|---------|
| `v2/subject/recent_hot/{kind}` | 热门推荐 | `kind`=movie/tv/show/anime |
| `v2/{kind}/recommend` | 带筛选推荐 | `tags` 逗号分隔，但 `tags=热门` 返回空！ |

⚠️ **豆瓣图片**：`img9.doubanio.com` 被拦截（418），必须替换为 `img3.doubanio.com` + Referer 头

#### 8.2.4 车机系统专项适配

> ⚠️ **车机适配是重点难点**，必须单独处理。

##### 车机交互特点

| 特点 | 说明 | 适配要求 |
|------|------|--------|
| **旋钮控制** | 部分车机用旋钮而非触摸 | 所有可交互元素必须支持焦点导航 |
| **驾驶模式** | 行驶中限制操作复杂度 | 大按钮、大文字、减少操作步骤 |
| **横屏为主** | 多数车机是横向宽屏 | 横向布局优先，16:9 或更宽 |
| **分心限制** | 驾驶时不能长时间看屏幕 | 3秒内完成核心操作，语音优先 |
| **色彩限制** | 车机屏幕可能偏暗/反光 | 高对比度，避免浅色文字 |
| **Android Automotive** | 部分车机原生系统 | 需要 Car App Library 兼容 |

##### 车机布局方案

#### 8.2.6 焦点导航（车机/遥控器必须）

```kotlin
// 所有可交互元素必须支持焦点
@Composable
fun FocusableCard(onClick: () -> Unit) {
    Card(
        modifier = Modifier
            .focusable()
            .onKeyEvent { event ->
                if (event.key == Key.Enter && event.type == KeyEventType.KeyUp) {
                    onClick()
                    true
                } else false
            }
            .clickable { onClick() }
    ) {
        // 内容
    }
}

// 焦点高亮样式
val focusedBorder = BorderStroke(3.dp, MaterialTheme.colorScheme.primary)
```

**焦点导航要求：**
- 所有 Card / Button / ListItem 必须支持焦点
- 焦点状态必须有明显视觉反馈（边框高亮 / 放大 / 阴影）
- 方向键 / 旋钮可导航到所有功能
- 焦点顺序：从左到右、从上到下
- 首页加载后默认焦点在「继续播放」按钮

### 8.4 封面自适应主题色

从封面图提取主色调，应用到详情页背景渐变：

```javascript
const extractDominantColor = async (imageUrl) => {
  const canvas = document.createElement('canvas')
  const ctx = canvas.getContext('2d')
  const img = new Image()
  img.crossOrigin = 'anonymous'
  img.src = imageUrl
  await new Promise(r => img.onload = r)
  canvas.width = 1; canvas.height = 1
  ctx.drawImage(img, 0, 0, 1, 1)
  const [r, g, b] = ctx.getImageData(0, 0, 1, 1).data
  return `rgb(${r},${g},${b})`
}
```

## 13. 前端开发注意事项

### 13.1 绝对不要踩的坑

| 坑 | 说明 | 后果 |
|----|------|------|
| **ExoPlayer HLS** | 必须有 `media3-exoplayer-hls` 依赖 | native 崩溃 |
| **LunaTV 搜索参数** | 用 `q` 不是 `keyword` | 搜索返回空 |
| **播放记录保存格式** | `{"key":"source+id","record":{...}}` | 保存无效 |
| **豆瓣图片** | `img9.doubanio.com` 被拦截 | 图片全白 |
| **豆瓣 recommend+tags=热门** | 返回空结果！ | 热门电影消失 |
| **LaunchedEffect 位置** | 不要放 LazyColumn item 内部 | 滚动时被回收 |
| **ExoPlayer seekTo** | prepare 后不能立即 seek | 必须等 STATE_READY |
| **源切换** | 不要依赖 LaunchedEffect 间接控制 | 直接控制 ExoPlayer |
| **LunaTV 返回格式** | 裸 JSON，没有 code/msg 包装 | 解析失败 |
| **播放记录格式** | `/api/playrecords` 返回 dict 不是 list | 遍历报错 |

### 13.2 ExoPlayer 核心用法

```kotlin
// ✅ 正确: 直接控制
exoPlayer.stop()
exoPlayer.setMediaItem(MediaItem.fromUri(url))
exoPlayer.prepare()
exoPlayer.playWhenReady = true

// ❌ 错误: 通过 ViewModel 状态间接触发（有延迟和竞争条件）
viewModel.switchSource(info)
```

### 13.3 搜索精确匹配

```kotlin
// ✅ 正确: 去空格 + 小写 + 精确比较
val normalizedTitle = title.replace(" ", "").lowercase()
val matched = sources.firstOrNull {
    it.title.replace(" ", "").lowercase() == normalizedTitle
    && it.year == targetYear
}

// ❌ 错误: contains 模糊匹配（会匹配到其他影片）
val matched = sources.firstOrNull { it.title.contains(title, ignoreCase = true) }
```

### 13.4 豆瓣图片处理

```kotlin
// 替换域名 + 添加 Referer
val safeUrl = originalUrl.replace(
    Regex("img\\d+\\.doubanio\\.com"),
    "img3.doubanio.com"
```
